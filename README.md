<h1 align="center">Gibil Agent Skill</h1>

<p align="center">
  Teach your AI agent how to use <a href="https://gibil.dev">gibil</a> — ephemeral remote servers for builds, tests, and deployments.
</p>

---

## Install

```bash
npx skills add https://github.com/AlexikM/gibil-skills --skill gibil
```

Works with Claude Code, Cursor, Copilot, Gemini CLI, and [40+ other agents](https://agentskills.io).

## Prerequisites

You need gibil installed:

```bash
npm install -g gibil
gibil init
```
